**Пользователь** *может*:
- Зарегестрироваться
- Просмотреть список товаров
- Оформить заказ
- Оплатить заказ 

**Админ** *может*:
- Получить уведомление об получении нового заказа
- Просмотреть даты заказов
- Просмотреть заказы за указаный день
- Просмотреть список пользователей для уточнения информации
